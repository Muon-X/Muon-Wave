### 上手指南

#### 要求

- KiCad 8.0
- Vivado
- MounRiver Studio

#### 安装步骤

1. 克隆库
```sh
git clone https://github.com/Muon-X/Muon-Wave.git
```
2. 使用要求的软件进行开发
